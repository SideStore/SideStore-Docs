# Contributing to the SideStore Documentation

Thank you for your interest in contributing to SideStore's Documentation!

you.

Certificate of Origin found in [CERTIFICATE-OF-ORIGIN.md](CERTIFICATE-OF-ORIGIN.md). Any contributions to
