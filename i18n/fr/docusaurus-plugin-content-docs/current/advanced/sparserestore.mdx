---
title: Contourner la limite des 3 applications
description: Un tutoriel sur la manière d'installer plus de 3 applications via SideStore en utilisant l'exploit SparseRestore.
---

:::note

L'exploit SparseRestore pour "contourner la limite des 3 applications" **ne fonctionne que sur les versions iOS/iPadOS entre 17.0 et 18.1 bêta 4 (hors 17.7.1)**. Si vous n'êtes pas sur ces versions et que cela ne fonctionne pas, merci de **ne pas demander de support à ce sujet**.

:::

## Prérequis
* Un appareil iOS/iPadOS fonctionnant sous une version compatible avec l'exploit SparseRestore (voir la note ci-dessus).
* Au moins 1 emplacement d'application libre.
* Version 0.5.9 de SideStore ou supérieure. Vous pouvez trouver les dernières versions [ici](https://github.com/SideStore/SideStore/releases).
* Pour effectuer cet exploit, vous DEVEZ désactiver **Localiser** sur votre appareil iOS pendant le processus. Sinon, cela ne fonctionnera pas. Voici comment vous pouvez le désactiver:
  - Ouvrez l'application Réglages.
  - Tapez sur votre nom en haut.
  - Tapez sur Localiser.
  - Tapez sur Localiser mon iPhone/iPad.
  - Désactiver Localiser mon iPhone/iPad.
  - Il se peut que vous soyez invité à entrer le mot de passe de votre identifiant Apple pour confirmer.
  
  > **Remarque:** Il se peut que vous ne puissiez toujours pas désactiver Localiser, car la Protection des appareils volés peut empêcher cette désactivation. Désactivez la Protection des appareils volés avant d'essayer de désactiver Localiser.

## Étapes
1. Installez SparseBox. Cela peut être fait de différentes manières, mais pour ce tutoriel, installez-le avec SideStore. La dernière version peut être trouvée [ici](https://nightly.link/khanhduytran0/SparseBox/workflows/build/main/artifact.zip) (décompressez artifact.zip pour obtenir le fichier .ipa).

2. Cochez "Désactiver la limite d'applications" dans SideStore, puis assurez-vous que SideStore est **fermé** (pas en arrière-plan), avant d'ouvrir SparseBox.

3. Activez StosVPN, sélectionnez votre fichier d'appairage et tapez sur 'Contourner la limite des 3 applications'. Cela devrait afficher une sortie de journal. Attendez que cela affiche `idevicebackup2 exited with code -18` ou `idevicebackup2 exited with code 2`, puis fermez l'application. Il n'est pas nécessaire de cliquer sur Appliquer.

4. Rouvrez SideStore, puis faites défiler les paramètres de SideStore. Appuyez sur "Vider le cache", cela permettra à l'exploit d'être visible par SideStore.

5. Testez si l'exploit a fonctionné (vous devrez peut-être installer plus d'une application pour tester, en fonction du nombre de slots disponibles). Si une erreur concernant la limite apparaît, rouvrez SideStore ou réessayez l'exploit. Si l'application s'installe, félicitations! Vous pouvez désormais avoir plus de 3 applications installées en même temps.

## Remarques
En raison de certaines limitations, cet exploit doit être effectué pour chaque groupe de 3 applications supplémentaires que vous installez.

Si vous souhaitez/utilisez Localiser pour d'autres raisons, vous pouvez maintenant le réactiver. Cependant, si vous souhaitez installer ou actualiser plus d'applications, vous devrez le désactiver à nouveau.

## Alternatives
Si l'exploit ne fonctionne pas pour vous, ou si votre appareil iOS est sur une version non prise en charge, il existe une alternative pour pouvoir utiliser plus de 3 applications en même temps. LiveContainer, développé par ([khanhduytran0](https://github.com/khanhduytran0)) (la personne qui a également créé SparseBox), permet de faire fonctionner des applications iOS dans des conteneurs au sein de l'application, plutôt que de les installer séparément. Cela peut également être plus facile à utiliser que SparseRestore dans certains cas. La page GitHub peut être trouvée [ici](https://github.com/khanhduytran0/LiveContainer).
