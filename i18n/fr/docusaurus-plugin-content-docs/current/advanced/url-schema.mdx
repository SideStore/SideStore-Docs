# Schéma d'URL

Voici la liste des différents schémas d'URL qui peuvent être utilisés pour interagir avec SideStore. Ils sont extrêmement utiles car ils permettent aux développeurs d'ajouter simplement un lien en un clic sur leurs réseaux sociaux ou sites web, et iOS ouvrira directement le lien dans SideStore, en effectuant l'action spécifiée.

La liste actuelle des actions que vous pouvez effectuer avec les schémas d'URL est limitée à l'installation d'applications et de sources, mais elle sera étendue à l'avenir pour accepter plusieurs entrées et effectuer une variété d'autres tâches.

## Installer une application hébergée à distance

```
sidestore://install?url=[URL de téléchargement]
```

## Ajouter une AltSource

```
sidestore://source?url=[URL source]
```
