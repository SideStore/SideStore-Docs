## Problèmes courants avec SideStore

### Échec de la connexion AFC
#### Cause
Problèmes de connexion ou fichier d'appairage invalide.
#### Résolution
1. Essayez de vous reconnecter
2. Générez un nouveau fichier d'appairage en utilisant JitterbugPair et importez-le via le menu des paramètres. Consultez nos [instructions pour le fichier d'appairage](/docs/installation/pairing-file) pour plus de détails.

### Pas de connexion Wi-Fi ou VPN
#### Cause
Problèmes de connexion avec le service minimuxer.
#### Résolution
1. Si vous utilisez WireGuard, passez à StosVPN
2. Essayez à nouveau (spammer le bouton aide généralement)
3. Redémarrez SideStore
4. Redémarrez StosVPN
5. Générez un nouveau fichier d'appairage en utilisant JitterbugPair et importez-le via le menu des paramètres. Consultez nos [instructions pour le fichier d'appairage](/docs/installation/pairing-file) pour plus de détails.

### SideStore se bloque à mi-chemin de l'installation
#### Cause
Problèmes de cache.
#### Résolution
1. Redémarrez SideStore
2. Videz le cache
3. Changez de serveur Anisette
4. Réinitialisez adi.pb
5. Déconnectez-vous de SideStore puis reconnectez-vous
6. Utilisez WireGuard
7. Redémarrez l'appareil
8. Régénérez le fichier d'appairage
9. Réinstallez SideStore

### Impossible de lancer SideStore
#### Cause
Plusieurs causes possibles.
#### Résolution
1. Assurez-vous d'avoir installé SideStore via AltServer.
2. Réinstallez SideStore

Si les étapes ci-dessus ne fonctionnent pas, faites ce qui suit:

1. Supprimez SideStore et toutes les applications que vous avez installées depuis SideStore.
2. Réinstallez SideStore.
3. Sélectionnez le fichier d'appairage et connectez-vous. (Un message apparaîtra pour actualiser SideStore, sélectionnez non)
4. Installez le fichier Sidestore.ipa sur votre appareil.
5. Importez le fichier Sidestore.ipa dans SideStore comme vous le feriez pour installer n'importe quelle autre application.
6. Cela permettra à SideStore de faire une actualisation propre, et espérons-le de résoudre le problème.
