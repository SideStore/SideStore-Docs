## Installation

#### J'ai suivi les instructions, mais je ne parviens toujours pas à installer SideStore.

### Instructions pour Mac
1. **Faire confiance à l'appareil**: Après avoir connecté votre appareil iOS à votre Mac, assurez-vous d'avoir sélectionné “Faire confiance” à la fois sur votre Mac et votre appareil iOS. Vous pouvez vérifier cela en:
   - Ouvrir Finder (macOS 10.15 Catalina ou version ultérieure) ou iTunes (macOS 10.14 Mojave et versions antérieures).
   - Déconnectez et reconnectez votre téléphone à votre Mac, puis vérifiez si une boîte de dialogue apparaît vous demandant si vous voulez faire confiance à l'appareil.
2. **Utiliser un autre identifiant Apple**: Essayez d'utiliser un autre identifiant Apple. Si nécessaire, vous pouvez [créer un nouvel identifiant Apple](https://appleid.apple.com/account#!\&page=create) spécifiquement pour SideStore, gratuitement.

### Instructions pour Windows
1. **Faire confiance à l'appareil**: Après avoir connecté votre appareil iOS à votre ordinateur Windows, assurez-vous d'avoir sélectionné “Faire confiance” à la fois sur votre ordinateur et votre appareil iOS. Vous pouvez vérifier cela en:
   - Ouvrir iTunes et vérifier si une boîte de dialogue apparaît vous demandant si vous voulez faire confiance à l'appareil.
2. **Exécuter AltServer en tant qu'administrateur**: Faites un clic droit sur AltServer et sélectionnez “Exécuter en tant qu'administrateur” pour garantir les autorisations nécessaires.
3. **Utiliser un autre identifiant Apple**: Essayez d'utiliser un autre identifiant Apple. Si nécessaire, vous pouvez [créer un nouvel identifiant Apple](https://appleid.apple.com/account#!\&page=create) spécifiquement pour SideStore, gratuitement.
4. **Version non-Microsoft Store d'iTunes et iCloud**:
   - Si vous avez installé iTunes ou iCloud depuis le Microsoft Store, vous devrez les désinstaller.
   - Téléchargez et installez les versions non-Microsoft Store d'iTunes et iCloud en utilisant les liens suivants:
     - [Télécharger iCloud](https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe)
     - [Télécharger iTunes](https://www.apple.com/itunes/download/win64/)


## Problèmes de connexion

#### SideStore se bloque ou affiche un code d'erreur lors de la connexion

Si vous rencontrez des blocages ou des codes d'erreur lors de la tentative de connexion à SideStore, plusieurs causes possibles peuvent être en jeu. Suivez ces étapes pour résoudre le problème:

1. **Changer le serveur Anisette:**
   La solution la plus courante consiste à passer à un autre serveur Anisette. Les serveurs Anisette peuvent parfois rencontrer des périodes d'indisponibilité. Passer à un serveur alternatif résout généralement le problème. Faites cela:
   - Ouvrez les paramètres de SideStore.
   - Faites défiler vers le bas jusqu'à l'option "Anisette Server".
   - Sélectionnez un autre serveur dans la liste.
   - Vous devrez peut-être essayer plusieurs serveurs de la liste pour trouver celui qui fonctionne le mieux pour vous.

2. **Hébergez votre propre serveur Anisette:**
   Si vous préférez, vous pouvez configurer votre propre serveur Anisette. Des instructions détaillées pour l'hébergement d'un serveur Anisette sont disponibles dans la documentation officielle et peuvent être consultées [ici](https://docs.sidestore.io/docs/advanced/anisette).

#### Code de vérification non reçu lors de la connexion avec l'identifiant Apple

Un autre problème courant lors de la connexion est de ne pas recevoir un code de vérification lorsque vous utilisez votre identifiant Apple avec SideStore. Cela peut être résolu en suivant ces étapes:

**Pour les versions iOS inférieures à 18.1:**

1. Ouvrez l'application "Réglages".
2. Tapez sur votre nom en haut de l'écran.
3. Allez dans "Identifiant Apple et sécurité".
4. Sélectionnez "Authentification à deux facteurs".
5. Choisissez "Obtenir le code de vérification".

**Pour les versions iOS 18.1 et supérieures:**

1. Rendez-vous sur [iCloud](https://www.icloud.com) dans un navigateur web.
2. Cliquez sur "Se connecter".
3. Lorsque vous y êtes invité, deux options s'afficheront: "Se connecter" et "Utiliser un autre identifiant Apple".
   - Sélectionnez l'option du bas : "Utiliser un autre identifiant Apple".
4. Saisissez votre identifiant Apple ainsi que votre mot de passe.
5. Apple vous enverra un code de vérification.
   - Utilisez ce code dans Sideload/Sidestore pour finaliser la connexion.


## Impossible d'actualiser ou d'installer des applications

#### Résolution du problème No Wi-Fi or VPN Error

1. Désactivez vos données cellulaires / connectez-vous à un réseau Wi-Fi stable.
2. Assurez-vous que les domaines Apple, ainsi que celui de votre serveur Anisette, ne sont pas bloqués. Vérifiez les points suivants:
   - Désactivez tout blocage DNS affectant des domaines comme `oscp.apple.com`
   - Déconnectez-vous du Wi-Fi d'un établissement scolaire ou professionnel, et essayez un réseau sans restrictions.
3. Vérifiez que le VPN est bien connecté dans StosVPN.
4. Désactivez puis réactivez StosVPN, attendez quelques secondes dans SideStore avant de tenter une actualisation.
5. Créez un nouveau fichier d'appairage.
   - Si aucune des étapes précédentes n'a fonctionné, il est fort probable que le fichier d'appairage soit corrompu. Vous pouvez consulter la documentation pour créer un nouveau fichier d'appairage [ici](/docs/installation/pairing-file).
   - Une fois le nouveau fichier créé, allez dans les paramètres de SideStore, appuyez sur "Reset pairing file", puis sélectionnez le fichier que vous venez de créer.

## Impossible de sélectionner le fichier d'appairage

#### Si vous ne parvenez pas à sélectionner un fichier d'appairage dans SideStore, suivez ces étapes pour résoudre le problème:

1. **Vérifiez l'extension du fichier**: Assurez-vous que l'extension de votre fichier d'appairage se termine par `.mobiledevicepairing` ou `.plist`. Si ce n'est pas le cas, vérifiez si vous aviez compressé le fichier avant de l'envoyer sur votre téléphone. Ne pas le faire peut entraîner une corruption du fichier pendant le transfert.
2. **Déplacez le fichier d'appairage**: Si vous ne parvenez pas à sélectionner le fichier depuis l'application, renommez-le en `ALTPairingFile.mobiledevicepairing`, puis essayez de le déplacer dans le dossier SideStore situé dans "Sur mon iPhone/iPad" via l'app Fichiers.
3. **Signature avec certificat**: Si vous avez signé SideStore avec un certificat, il se peut que vous ne puissiez pas sélectionner le fichier d'appairage depuis l'application. Vous pouvez essayer la méthode mentionnée ci-dessus. Si le dossier SideStore n'apparaît pas dans l'app Fichiers, connectez votre appareil à un ordinateur et glissez-déposez le fichier d'appairage dans les fichiers de l'application SideStore. Assurez-vous de bien le renommer comme indiqué précédemment.

## AltServer

#### AltStore affiche “Could not find AltServer” lors de l'installation ou de l'actualisation d'applications.

AltServer doit être lancé sur un ordinateur connecté au même réseau Wi-Fi qu'AltStore afin de pouvoir installer ou actualiser des applications. Si AltServer est bien en cours d'exécution sur le même réseau Wi-Fi qu'AltStore et que vous recevez toujours cette erreur, essayez les solutions suivantes:

- Êtes-vous connecté à un réseau Wi-Fi public/professionnel/scolaire? Si c'est le cas, ce type de réseau peut empêcher les appareils de se détecter mutuellement. Essayez de connecter votre ordinateur au point d'accès de votre téléphone, puis réessayez. (Si vous utilisez un iPad, assurez-vous qu'il est lui aussi connecté à votre point d'accès.)
- Avez-vous cliqué sur "Faire confiance" sur votre Mac et votre appareil iOS après les avoir connectés? Pour vérifier, ouvrez le Finder (macOS 10.15 Catalina ou version ultérieure) ou iTunes (macOS 10.14 Mojave ou version antérieure) et voyez si une boîte de dialogue s'affiche vous demandant si vous faites confiance à cet appareil.
- (Windows) Il se peut que votre pare-feu bloque les connexions réseau entrantes vers AltServer. Vous devez autoriser l'accès réseau à AltServer dans les paramètres de votre pare-feu pour que les applications puissent être actualisées via AltStore.
- (Windows) Avez-vous installé iTunes ou iCloud via le Microsoft Store? Si oui, vous devez les désinstaller et télécharger les dernières versions directement depuis le site d'Apple.
- Enfin, essayez de connecter votre iPhone ou iPad directement à votre ordinateur via câble USB. Cela devrait résoudre tous les problèmes de connectivité, mais AltStore ne pourra alors plus actualiser automatiquement les applications en arrière-plan via le Wi-Fi.

#### Je ne peux pas modifier les paramètres réseau pour permettre aux appareils de communiquer entre eux (par exemple sur un Wi-Fi scolaire/professionnel/public).

Vous pouvez toujours installer et actualiser des applications sans utiliser le Wi-Fi en connectant votre iPhone ou iPad à votre ordinateur via un câble. Cependant, cela signifie qu'AltStore ne pourra pas actualiser automatiquement les applications en arrière-plan via le Wi-Fi.

## AltJIT

#### AltServer affiche “JIT could not be enabled for [App]".

Cela peut se produire pour plusieurs raisons. Si vous rencontrez ce problème, veuillez vérifier les éléments suivants:

- Assurez-vous que l'application pour laquelle vous essayez d'activer JIT est bien en cours d'exécution au premier plan sur votre appareil.
- Êtes-vous connecté à un réseau Wi-Fi public/professionnel/scolaire? Si oui, il se peut que ce réseau empêche les appareils de se détecter entre eux. Essayez de connecter votre ordinateur au point d'accès de votre téléphone, puis réessayez. (Si vous utilisez un iPad, assurez-vous qu'il est également connecté à votre hotspot.)
- Avez-vous cliqué sur "Faire confiance" sur votre Mac et sur votre appareil iOS après les avoir connectés? Pour vérifier, ouvrez le Finder (macOS 10.15 Catalina ou version ultérieure) ou iTunes (macOS 10.14 Mojave ou version antérieure), et voyez si une boîte de dialogue vous demande si vous faites confiance à cet appareil.
- (Windows) Le pare-feu peut bloquer les connexions réseau entrantes vers AltServer. Vous devez autoriser l'accès réseau à AltServer dans les paramètres de votre pare-feu pour que les applications puissent être actualisées ou que JIT fonctionne.
- (Windows) Avez-vous installé iTunes ou iCloud via le Microsoft Store? Si c'est le cas, vous devez les désinstaller, puis télécharger les dernières versions directement depuis le site d'Apple.
- Assurez-vous également qu'iTunes et iCloud sont bien en cours d'exécution sur votre ordinateur.
- Enfin, essayez de connecter votre iPhone ou iPad à votre ordinateur via un câble USB. Cela devrait résoudre tous les problèmes de connectivité, mais cela signifie qu'AltStore ne pourra pas actualiser automatiquement les apps ou activer JIT en arrière-plan via le Wi-Fi.

#### Que faire si je dois utiliser la version Microsoft Store d'iCloud?

AltStore nécessite que vous installiez iCloud directement depuis le site d'Apple afin de pouvoir authentifier votre identifiant Apple. Si vous devez utiliser la version d'iCloud disponible sur le Microsoft Store, vous pouvez suivre ces instructions alternatives, reprises de [ce post Reddit](https://www.reddit.com/r/jailbreak/comments/rp5buz/tutorial\_how\_to\_use\_altserver\_if\_you\_have\_to\_use/?utm\_source=share\&utm\_medium=ios\_app\&utm\_name=iossmf):

1. Téléchargez et installez [iTunes](https://www.apple.com/itunes/) depuis le site officiel d'Apple. Veillez à faire **défiler la page vers le bas** et à cliquer sur "Windows" à côté du texte "Vous cherchez d'autres versions?".
2. Téléchargez et installez [iCloud](https://support.apple.com/en-us/HT204283) depuis le site officiel d'Apple. Faites également **défiler la page vers le bas** et cliquez sur le lien contenu dans ce texte: "\* Sur Windows 7 et Windows 8, vous pouvez _télécharger iCloud pour Windows sur le site d'Apple_".
3. Allez dans le dossier "C:\Program Files (x86)\Common Files\Apple". Copiez les dossiers Apple Application Support et Internet Services, puis créez un nouveau dossier (vous pouvez lui donner le nom que vous voulez) et collez-les dedans.
4. Désinstallez iCloud de votre ordinateur. **NE DÉSINSTALLEZ PAS** iTunes! (De plus, **NE SUPPRIMEZ JAMAIS** les éléments "Apple Mobile Device Support" ou "Apple Application Support" installés avec iTunes et iCloud.)
5. Installez [iCloud](https://www.microsoft.com/store/productId/9PKTQ5699M62) depuis le Microsoft Store.
6. Téléchargez et installez [AltServer](https://altstore.io/). Lors du premier lancement, cliquez sur "Choose Folder...", puis sélectionnez le dossier que vous avez créé à l'étape 3.
