Pour installer SideStore (ou toute autre application IPA) sous Linux, assurez-vous que Docker est installé. Avant de commencer, installez StosVPN ainsi que toutes les [prérequis](prerequisites) nécessaires!

1. Exécutez altcon avec Docker
```bash
docker run --rm -it -e ALTSERVER_ANISETTE_SERVER="https://ani.sidestore.io/" -v ${PWD}/:/mnt/ -v /var/run:/var/run ghcr.io/sidestore/altcon
```

Vous pouvez également exécuter altcon avec Podman
```bash
podman run --privileged --rm -it -e ALTSERVER_ANISETTE_SERVER="https://ani.sidestore.io/" -v ${PWD}/:/mnt/ -v /var/run:/var/run ghcr.io/sidestore/altcon
```

Altcon commencera à s'apparier avec votre appareil. Branchez-le si ce n'est pas déjà fait, appuyez sur "Faire confiance" et saisissez votre code d'accès.

Une fois terminé, suivez les [instructions après l'installation](post-install).
