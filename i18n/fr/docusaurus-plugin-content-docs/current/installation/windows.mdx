title: Installation sous Windows

Assurez-vous que les versions non-Microsoft Store d'[iCloud](https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe) et [iTunes](https://support.apple.com/en-us/HT210384) sont installées sur votre machine, si vous utilisez Windows. Avant de commencer, assurez-vous d'avoir complété toutes les étapes des [prérequis](prerequisites) et d'avoir un [fichier d'appairage](pairing-file)!

La version non-Microsoft Store d'iCloud a été retirée du site web d'Apple vers mi-septembre 2022. Le lien ci-dessus dirige vers une version plus ancienne d'iCloud qui reste compatible et est toujours disponible directement depuis les serveurs d'Apple.

1. Téléchargez et exécutez AltServer depuis le [site web d'AltStore](https://altstore.io/).
2. Téléchargez le fichier IPA de SideStore que vous pouvez trouver [ici](https://github.com/sidestore/sidestore/releases/latest/download/sidestore.ipa)
3. Maintenez la touche Maj enfoncée et cliquez sur l'icône d'AltServer dans la barre d'état système. Cliquez sur "Sideload .ipa" et sélectionnez le fichier IPA de SideStore.
4. Approuvez l'"Application développeur" avec l'adresse e-mail de votre identifiant Apple.
    * Vous pouvez le trouver dans les Réglages, sous "Général", puis "VPN et gestion de l'appareil".
5. Ouvrez SideStore, sélectionnez votre fichier d'appairage et connectez-vous avec votre identifiant Apple.

Une fois terminé, suivez les [instructions après l'installation](post-install).
