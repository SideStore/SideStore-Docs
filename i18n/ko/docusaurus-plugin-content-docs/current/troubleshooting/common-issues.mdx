## SideStore의 일반적인 문제

### AFC Connection Failure
#### 발생 원인
연결 문제 또는 페어링 파일이 잘못되었습니다.
#### 해결 방법
1. 재연결 시도
2. JitterbugPair를 사용하여 새 페어링 파일을 생성하고 설정 메뉴를 통해 가져옵니다. 자세한 내용은 [페어링 파일 지침](/docs/installation/pairing-file)을 참조하세요.

### No WiFi or VPN Connection
#### 발생 원인
미니먹서 서비스의 연결 문제.
#### 해결 방법
1. WireGuard를 사용 중인 경우, StosVPN으로 전환하세요.
2. 다시 시도하세요 (보통 버튼을 연타하면 도움이 됩니다.)
3. SideStore를 다시 시작하세요.
4. StosVPN 재시작
5. JitterbugPair를 사용하여 새 페어링 파일을 생성하고 설정 메뉴를 통해 가져옵니다. 자세한 내용은 [페어링 파일 지침](/docs/installation/pairing-file)을 참조하세요.

### 설치 도중에 SideStore가 멈춤
#### 발생 원인
캐시 문제.
#### 해결 방법
1. SideStore를 다시 시작합니다.
2. 캐시 지우기
3. Anisette 서버 변경
4. adi.pb 재설정
5. SideStore에서 로그아웃하고 다시 로그인합니다.
6. WireGuard 사용
7. 기기 다시 시작
8. 페어링 파일 재생성
9. SideStore를 다시 설치합니다.

### SideStore를 열 수 없습니다.
#### 발생 원인
여러 가지 원인이 있을 수 있습니다.
#### 해결 방법
1. AltServer를 통해 SideStore를 설치했는지 확인합니다.
2. SideStore를 재설치합니다.

위의 방법으로 해결되지 않으면 다음을 수행하세요:

1. SideStore 및 SideStore에서 설치한 모든 앱을 삭제합니다.
2. SideStore를 다시 설치합니다.
3. 페어링 파일을 선택하고 로그인합니다. (SideStore를 새로고침하라는 메시지가 표시되면 아니요를 선택합니다.)
4. SideStore.ipa 파일을 기기에 설치합니다.
5. 다른 앱을 설치할 때와 마찬가지로 SideStore.ipa 파일을 SideStore로 가져옵니다.
6. 이렇게 하면 SideStore가 새로고침을 수행하여 문제가 해결될 수 있습니다.
