시작하기 전에, [설치 전 준비하기](prerequisites)에 있는 모든 소프트웨어를 다운로드하고 설정을 완료하세요!

## SideStore 설치하기
### 컴퓨터에서
1. 당신의 iPhone, iPad 또는 iPod touch를 케이블을 통해 컴퓨터에 연결해 주세요. 만약 신뢰 여부를 묻는 창이 뜬다면, 신뢰 버튼을 누르고 기기 비밀번호를 입력해 주세요.
2. iloader를 열어주세요.
3. 애플 계정으로 로그인을 해주세요. (연결된 당신의 기기와 계정이 똑같을 필요는 없어요. 대소문자를 구분한다는 점을 기억하세요!)
4. 당신의 iPhone, iPad 또는 iPod touch를 선택해 주세요.
5. 'Install SideStore (Stable)'를 선택해 주세요.

### iPhone, iPad 또는 iPod touch에서

이 과정이 성공하면, SideStore가 당신의 기기를 홈 화면으로 보내주고 알림도 뜰 겁니다. 몇 초 안에 SideStore를 다시 사용할 수 있어요.

주의: iPhone, iPad 또는 iPod touch를 업데이트하거나 초기화하면 페어링 파일이 만료돼서, 다시 가져와야 해요. 이 방법은 [이 가이드](/docs/advanced/pairing-file)를 보면 돼요. 이건 애플 소프트웨어 때문에 가끔 무작위로 만료될 수 있어요.
