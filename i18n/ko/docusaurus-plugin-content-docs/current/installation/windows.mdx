Microsoft 스토어에서 설치되지 않은 [iCloud](https://updates.cdn-apple.com/2020/windows/001-39935-20200911-1A70AA56-F448-11EA-8CC0-99D41950005E/iCloudSetup.exe) 및 [iTunes](https://support.apple.com/en-us/HT210384)가 설치되어 있는지 확인하세요. 시작하기 전에 [필수 요구 사항](prerequisites)에 있는 모든 단계를 완료하고 [페어링 파일](pairing-file)을 준비했는지 확인하세요!

Microsoft 스토어가 아닌 버전의 iCloud는 2022년 9월 중순에 Apple 웹사이트에서 삭제되었습니다. 위의 링크는 여전히 호환되며 Apple 서버에서 직접 사용할 수 있는 이전 버전의 iCloud로 연결됩니다. 

1. AltServer를 [AltStore의 웹사이트](https://altstore.io/)에서 다운로드한 후 실행하세요.
2. [여기](https://github.com/sidestore/sidestore/releases/latest/download/sidestore.ipa)에서 SideStore IPA를 다운로드하세요.
3. Shift를 누른 상태로 AltServer 트레이 아이콘을 누르세요. 그 후 "Sideload .ipa"를 누른 후 다운로드받은 SideStore IPA를 넣으세요.
4. Apple 계정의 이메일로 "개발자 앱"을 승인하세요.
    * 설정 -> 일반 -> VPN 및 기기 관리에서 승인할 수 있습니다.
5. SideStore를 실행시킨 후, 페어링 파일을 선택하고 당신의 Apple 계정으로 로그인하세요.

완료되면 [설치 후 지침](post-install)을 따르세요.
