Um SideStore (oder auch andere IPAs) unter Linux zu installieren, stelle sicher, dass Docker installiert ist. Bevor du beginnst, stelle sicher, dass StosVPN installiert ist und alle anderen und [Voraussetzungen](prerequisites) erfüllt sind!

1. Führe altcon mit Docker aus
```bash
docker run --rm -it -e ALTSERVER_ANISETTE_SERVER="https://ani.sidestore.io/" -v ${PWD}/:/mnt/ -v /var/run:/var/run ghcr.io/sidestore/altcon
```

Du kannst altcon auch mit Podman ausführen
```bash
podman run --privileged --rm -it -e ALTSERVER_ANISETTE_SERVER="https://ani.sidestore.io/" -v ${PWD}/:/mnt/ -v /var/run:/var/run ghcr.io/sidestore/altcon
```

Altcon wird anfangen sich mit deinem Gerät zu verbinden. Schließe es an falls du das noch nicht getan hast, Drücke „Vertrauen“ und gib deinen Code ein.

Wenn du fertig bist, folge den [Anweisungen nach der Installation](post-install).
