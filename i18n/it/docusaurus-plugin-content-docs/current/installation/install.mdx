Prima di iniziare, assicurati di installare tutto il software e completare gli step della [pagina prerequisiti](prerequisites)!

## Installare SideStore
### Sul tuo Computer
1. Collega il tuo iPhone, iPad o iPod Touch al computer tramite cavo USB. Se te lo chiede, autorizza il computer e inserisci il codice.
2. Apri iloader.
3. Accedi col tuo Account Apple (non è necessario che sia lo stesso associato al tuo dispositivo. Ricordati che è case-sensitive!)
4. Seleziona il tuo iPhone, iPad o iPod Touch.
5. Seleziona 'Install SideStore (Stable)'.

Se tutto funziona correttamente, SideStore dovrebbe portarti alla home screen (e mostrare una notifica). SideStore sarà di nuovo disponibile dopo qualche secondo.

Il pairing file potrebbe scadere se aggiorni o formatti il tuo iPhone, iPad o iPod Touch. Se succede dovrai rimetterlo seguendo [questa guida](/docs/advanced/pairing-file). Questo potrebbe accadere anche a random a causa di Apple.
