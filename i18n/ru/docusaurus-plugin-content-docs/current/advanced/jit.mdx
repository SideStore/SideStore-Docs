---
title: Включение JIT
description: Инструкция по включению Just-in-Time (JIT) компиляции для приложений, использующих её.
---

:::note

Есть много способов для включения JIT, таких как AltJIT, SideJITServer и других. В этой инструкции мы будем использовать StikJIT/StikDebug, мы рекомендуем использовать этот метод. Или как альтернативу, SideStore 0.6.2-nightly.2025.04.15 имеет обновления в minimuxer, в которых имеется встроенное включение JIT, схожим принципом на StikDebug. Для его использования, подключитесь к StosVPN и Wi-Fi или выключите сотовые данные с Wi-Fi.
:::

## Требования
* Устройство на iOS/iPadOS 17.4 или выше, исключая 18.4b1. Если Вы на 17.0-17.3, используйте другие методы, по типу [SideJITServer](https://github.com/nythepegasus/SideJITServer)
* 1 бесплатный слот приложения для StikJIT

## Установка StikJIT через SideStore
   
1. **Загрузите StikJIT:**  
   Добавьте источник приложений в SideStore [здесь](sidestore://source?url=https://raw.githubusercontent.com/StephenDev0/StikJIT/refs/heads/main/repo.json) (Нужно чтобы SideStore был установлен, иначе ссылка не будет работать).

2. **Установите:**
   Используйте SideStore для установки StikJIT.

3. **Процедура установки:**
   Следуйте стандартной процедуре установки по инструкции ниже.

## Установка StikDebug через App Store (Не требует слот приложения)

1. **Установите StikDebug**
   Установите StikDebug с App Store [здесь](https://apps.apple.com/us/app/stikdebug/id6744045754). Приятный бонус - VPN, используемый в StikDebug схожий на тот, который используется в SideStore. Это означает, что Вы можете использовать его также для установки и освежение приложений. 

2. **Процедура установки:**
   Следуйте стандартной процедуре установки по инструкции ниже.

### Стандартная процедура установки:

1. **Включите VPN:**  
   Начните с включения StosVPN для StikJIT или встроенного VPN в StikDebug.

2. **Выгрузите файл сопряжения:**  
   При появлении запроса, импортируйте тот-же самый **разархивированный файл сопряжения** получаемый через JitterbugPair, что используется в SideStore. Для более детальной информации по получению файла сопряжения, смотрите [здесь](/docs/installation/pairing-file).

## Чтобы включить JIT:

1. **Включите VPN:**  
   Начните с включения StosVPN для StikJIT или встроенного VPN в StikDebug.
   
2. **Включите JIT:**  
   Нажмите на кнопку “Enable JIT” и выберите приложения из списка для включения JIT

### Офлайн включения JIT 
Для использования JIT без сети, включите либо StosVPN, либо StikDebug. Далее, выключите и Wi-Fi, и сотовые данные когда будете включать StikJIT (используйте Авиарежим). Теперь Вы можете включить JIT как делаете это обычно.
