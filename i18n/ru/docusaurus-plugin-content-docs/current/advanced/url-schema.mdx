# Схемы URL

Ниже приведен список различных схем URL, с помощью которых можно взаимодействовать с SideStore. Они очень полезные, так как позволяют разработчикам вписывать лишь одну ссылку в их социальных сетях или сайтах, и iOS откроёт ссылку напрямую в SideStore, выполняя данные задачи.

В данный момент список того, что Вы можете сделать с URL схемами ограничен установкой приложений и источников приложений, но в будущем планируется добавить возможность принятия нескольких входных данных и выполнения множество других задач.
## Установка удалённо размещённого приложения

```
sidestore://install?url=[download url]
```

## Добавление источника приложений

```
sidestore://source?url=[source url]
```
