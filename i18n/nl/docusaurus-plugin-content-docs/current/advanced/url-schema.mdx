# URL Schema

Het volgende is de lijst met verschillende URL-schema’s die gebruikt kunnen worden om met SideStore te communiceren. Deze zijn enorm handig omdat ontwikkelaars hiermee een éénklik-link op hun sociale media of website kunnen plaatsen, waarna iOS de link direct in SideStore opent en de opgegeven actie(s) uitvoert.

De huidige lijst met mogelijkheden binnen URL-schema’s is beperkt tot het installeren van apps en bronnen (sources), maar deze zal in de toekomst worden uitgebreid om meerdere invoeren te ondersteunen en een bredere reeks taken uit te voeren.

## Een extern gehoste app installeren

```
sidestore://install?url=[download url]
```

## Een AltSource toevoegen

```
sidestore://source?url=[source url]
```
