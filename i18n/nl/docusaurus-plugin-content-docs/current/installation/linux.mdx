Om SideStore (of een andere IPA) op Linux te installeren, zorg je ervoor dat Docker is geïnstalleerd. Voordat je begint, installeer StosVPN en zorg dat alle andere [vereisten](prerequisites) zijn voldaan!

1. Voer altcon uit met Docker
```bash
docker run --rm -it -e ALTSERVER_ANISETTE_SERVER="https://ani.sidestore.io/" -v ${PWD}/:/mnt/ -v /var/run:/var/run ghcr.io/sidestore/altcon
```
Je kunt altcon ook uitvoeren met Podman

```bash
podman run --privileged --rm -it -e ALTSERVER_ANISETTE_SERVER="https://ani.sidestore.io/" -v ${PWD}/:/mnt/ -v /var/run:/var/run ghcr.io/sidestore/altcon
```

Altcon zal beginnen met koppelen aan je apparaat; sluit het aan als je dat nog niet hebt gedaan. Druk op “trust” en voer je toegangscode in.

Wanneer je klaar bent, volg dan de post-install instructies.
